# Introduction

GNSS(Global Navigation Satellite System)은 인공위성을 활용하는 항법이다.

- GPS는 현재 32대의 위성으로 별자리 구성
- 위성 4대의 신호를 받으면 필요한 항법 정보를 얻을 수 있으나, 일반적으로 최소 6~8대 정도 확보
- 위성의 수명은 대략 10년 내외이며, 주기적으로 위성을 발사하여 운영함
